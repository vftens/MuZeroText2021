**** MuZeroText
1) Idea was in new Program MuZeroText, that consists of additional Java Text module,
to provide text understanding features, with 3 neuro net modules of standard MuZero, that is to
provide MCTS search for solving the applied task of the Games.
2) Идея - попробовать развивать общение с пользователем через "additional Java Text module",
который позволит через MuZeroText общаться с пользователем и решать задачи в электронном магазине типа 
Shopify. Способность к анализу и оценке позволит выбирать из многих альтернатив при поиске решения.
Множество накопленных фраз Ф в "Java Text module" соотносится с 3-мя питоновскими модулями MuZero, которые
управляют процессом общения для достижения результата, который требуется в задаче.
